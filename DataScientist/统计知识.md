# 统计知识总结

# 1. 假设检验Hypothesis Test

假设实验的基本思想：小概率反证法，假设Null Hypothesis真实存在，那么我们在做sample test的时候遇到它不存在的概率一定很低。

## 1.1 分类：参数检验与非参数检验

  参数检验：parametric test, 已知数据为某一种分布。 \
  非参数检验：未知数据分布情况。

## 1.2 四种常见检验方法

  1. **t检验：（student）t Test**

      条件：
      样本量小于30 \
      总体标准差未知 \
      正态分布

      **单样本T检验：** 判断样本平均值与总体平均值（或给定平均值）是否存在显著差异。样本分布服从正态分布 \
      **配对T检验：** 判断两组对象（通常在不同条件下，如两个受试对象分别接受不同处理）是否有显著差别。差值需服从正态分布 \
      **双独立样本T检测：** 判断两组对象均值的差异，两组样本分布均服从正态分布。

  2. **z检验 z Test**   

      条件：
      样本量大于30 \
      总体方差已知

      在整体标准差已知的情况下比较总体平均值与某一值是否有显著差异。

  3. **卡方检验 Chi-Square Test**

    非参数检验，用来检验两个变量之间是否有关系

  4. **F检验 F Test**

    用来检验两个正态分布的总体是否具有方差齐性，后被应用于ANOVA TEST（检验是否所有sample都是random的）

## 1.3 其他检验方法

  1. **秩和检验**

    当两个样本来自于两个独立的但是非正态分布或者分布不明确的总体，检验样本中差异是否显著时不用t-test而用秩和检验。

    秩的定义：rank

    举例：

    一组数据[1,3,5,4] \
    排序后：[1,3,4,5] \
    3的秩：2

    检验方法：

    步骤一：将两份样本（设大小分别是n1, n2）混合并由小到大进行排列 \
    步骤二：把容量较小的样本中的数据的秩相加，得到检验值T \
    步骤三：根据n1 n2差检验表中T的上下限，如果超出上下限则差异显著

    [例子链接](https://wiki.mbalib.com/zh-tw/%E7%A7%A9%E5%92%8C%E6%A3%80%E9%AA%8C)

## 1.4 显著性检验

显著性检验背后的原理：小概率事件实际不可能性原理

  1. 一类错误 & 二类错误：

      一类错误：原假设（null hypothesis）为真时，决定放弃原假设，通常称其发生概率为α
      二类错误：原假设为假，决定不放弃原假设，通常称其发生概率为β。

  2. 显著性检验通常只限定α。如果p值小于某个水平，理论上拒绝零假设。
  3. 一般流程：

      1) 提出虚无假设和备择假设 \
      H0：______ \
      H1：______ \
      同时，与备择假设相应，指出所作检验为双尾检验还是左单尾或右单尾检验。

      2) 构造检验统计量，收集样本数据，计算检验统计量的样本观察值。

      3) 根据所提出的显著水平 ，确定临界值和拒绝域

      4) 计算检验统计量的值。

      5) 作出检验决策
  4. p值的定义：原假设(null hypothesis)为真时，实验样本观察结果或者更极端的结果出现的概率。

# 2. 各种典型的分布

## 2.1 伯努利分布 Bernoulli distribution

  定义：一种离散分布模型，伯努利实验有成功和失败两种可能，成功的概率为p 失败的概率为q=1-p

  概率质量函数：f(1) = p, f(0) = q \
  期望值：E = p \
  方差：Var = p\*q

## 2.2 二项分布 Binomial distribution

  定义：n个是/非实验成功的次数的离散概率分布，每次实验成功的概率为p。当n=1时，为伯努利分布。

  实验次数为n，成功概率为p时，写成 X ~ B(n,p)

  概率质量函数：(k: n次实验中成功k次) \
  f(k, n, p) = Pr(X = k) = (n choose k) * p^k * (1-p)^(n-k) \
  期望值：E = n * p \
  方差：Var = n * p * (1-p) = n * p * q

## 2.3 超几何分布 Hypergeometric Distribution

  定义：已知在N个样本中有K个符合某一条件，从N个样本中抽取n个（不放回），符合K条件的个数k的分布。如果条件改成可以放回，则变成二项分布。如条件改成n = 1，则变回伯努利分布。

  概率质量函数：f(k, N, K, n) = (K choose k) * (N-K choose n-k) / (N choose n)

## 2.4 几何分布 Geometric Distribution

定义：已知一个伯努利试验成功概率为p，重复这个实验，第一次成功的出现在的次数的分布

  概率质量函数：f(k, p) = (1-p)^k * - p \
  期望值：E = 1-p / p \
  方差：Var = 1 - p / p^2

## 2.5 负二项分布 Negative Binomial Distribution

  定义：重复一个成功概率为p的伯努利实验，出现r次成功前失败的次数。\
  可以用 X ~ NB(r, p)来表示

  概率质量函数：f(k; r, p) = Pr(X = K) = (k + r - 1 choose k) * p^r * (1-p)^k

## 2.6 泊松分布 Poisson Distribution

  定义：在单位时间内，发生频率为lambda的某一件事情发生k次的概率

  概率质量函数：f(k, lambda) = lambda^k * e^(-lambda) / k! \
  期望值；lambda \
  方差：lambda

## 2.7 平均分布 Uniform Distribution

  定义：均匀分布在a~b区间内

## 2.8 指数分布 Exponential Distribution

  定义：某一连续发生且发生p平均频率固定的时间发生间隔的分布

  概率密度函数：λ * e^(-λ * x) \
  期望值：1 / λ \
  方差：1 / λ²

## 2.9 正态 / 高斯分布 Normal / Gaussian Distribution

  定义： X ~ (μ，σ²) \
  μ：均值 \
  σ：标准差

# 3. 贝叶斯公式

    P(A|B) = P(B|A)P(A) / P(B)

    P(A|B): 后验概率 \
    P(A)：先验概率 \
    P(B|A)：似然性 \
    P(B)：标准化常量 \
    P(B|A) / P(B): 标准化似然性


    文字理解：后验概率 = 似然性 * 先验概率 / 标准化常量

# 4. 大数定律和中心极限定理


## 4.1 大数定律

  定义：实验的次数越多或者样本量越大，样本均值就回趋近于整体期望。

  重要性：大数定律说明了随机事件均值的长期稳定性

## 4.2 中心极限定理

  定义：对于任何分布的总体，其样本均值的分布接近于总体均值且围绕总体均值呈正态分布。

  重要性：证明了在没有办法得到总体数据的情况下可以用样本数据来估计总体数据。同时中心极限定理可以用来判断某一样本是否属于某一总体。

## 4.3 小数定律

  定义：把大样本中的结论错误移植到小样本的现象。

  举例：抛一个硬币，5次都是正面，猜测第六次是反面的概率较高。

### 1. 抛硬币问题

两人抛硬币，谁先扔出来正面谁赢，假如是你先扔硬币，那么赢的概率是多少？

```
答案：2/3

解答：

假设先扔的一方赢得概率为x，两方总有一个人赢，是必然事件，因此有

x+0.5*x=1，x=2/3

等式的物理意义是我赢的概率(x) 加上我第一次没正面的概率(0.5)，在我失败的情况下对方态势转化为我。
```

### 2. 赛马比赛

25匹马赛跑，5条跑道，跑几次可以知道前3？

```
答案：7次

解答：

1-5次：25匹马分成5组ABCDE，分别跑出最快的A1, B1, C1, D1, E1

6次：A1, B1, C1, D1, E1一起跑，跑出第一名，假设A1

7次：此时第二名很明显只可能是B1或者A2，因此只需要这两个赛跑就可以得到第二名

接着考虑如何用剩下的赛道选出第3：

如果第二名为A2，第三有可能是A3或B1

如果第二名为B1，第三名有可能是A2或者B2或者C1

以下是可能成为2、3名的马匹，正好5匹马，可以分成一组赛跑

[01, A2, A3, --, --]
[B1, B2, --, --, --]
[C1, --, --, --, --]
[--, --, --, --, --]
[--, --, --, --, --]

因此将[A2, A3, B1, B2, C1]一起赛跑可以得到二三名

```

64匹马赛跑，8条跑道，跑几次可以知道前4？

```
答案：11次

解法：

1-8次：64匹马分成8组，分别跑出：A1 ~ H1

9次：A1 ~ H1赛跑，淘汰最后四位：假设最快的是A1 B1 C1 D1

10次：此时一直A1是最快的，可以不用再跑A1，剩下A2 A3 A4 B1 B2 B3 C1 C2 D1
有可能进前4所以第10次从这9个里面抽8个跑，选出前三

11次：第10次跑出来的前三再和剩下的那个跑一次，再选出前三，加上A1就是前四
```

25匹马5条跑道找最快的5匹马，需要跑几次？

```
答案：8-9次

解法：

1-5次：同上，分成5组跑

假设结果：

[A1, A2, A3, A4, A5]
[B1, B2, B3, B4, B5]
[C1, C2, C3, C4, C5]
[D1, D2, D3, D4, D5]
[E1, E2, E3, E4, E5]

6次：同上，每组的第一名比赛

假设结果为A1最快，E1最慢

7次：

根据25匹马 5跑道 选前3，可以知道7轮可以选出前3

8次：

第7次的赛跑情况可能有以下几种结果，此处需要枚举分析所有可能性：

【可能性1：A2第二 A3第三】：

第四可能是 A4 或 B1

第四是A4: 第五可能是A5或B1
第四是B1：第五可能是A4或B2或C1

以下是可能成为4、5名的马，正好5匹，可以一次性决出四五名

[01, 02, 03, A4, A5]
[B1, B2, --, --, --]
[C1, --, --, --, --]
[--, --, --, --, --]
[--, --, --, --, --]

【可能性2：A2第二 B1第三】：

第四可能是A3, B2, C1

假设第四A3，第五可能是A4或B2或C1
假设第四B2, 第五可能是A3或B3或C1
假设第四C1，第五可能是A3或B2或C2或D1

综上：第4、5名需要在马匹[A3、B2、B3、C1、A4、C2、D1]七匹马中产生，则必须比赛两场才行，也就是到第9场角逐出全部的前5名。

【可能性3 - 5以此类推】

结论：需要 8 - 9场可以选出前5

```

### 3. 砝码称轻重

一共有9个砝码，一个天平，已知其中有1个砝码比其他的轻，称几次可以找到这个砝码？

```
答案：2次

解法：

第一次：

先分成3个为一组的3组，任意选两组放在天平上。

如果不一样重，轻的那组里面有要找的砝码。如果一样重，较轻的砝码在剩下的那组里。

第二次：

从找到的组里选两个放在天平上。

如果不一样重，轻的那个是要找的砝码。如果一样重，较轻的砝码是剩下那个。

```

有十组砝码每组十个，每个砝码重10g，其中一组每个只有9g，有能显示克数的秤最少几次能找到轻的那一组砝码？

```
答案：1次

解法：

10组砝码，第一组拿1个，第二组拿2个，以此类推第10组拿10个放在称上。

假设称的读数为 X，550-X就是每个只有9g的那一组
```

### 4. 牛奶咖啡

```

### 白鼠与毒药

有1000个一模一样的瓶子，其中有999瓶是普通的水，有1瓶是毒药。任何喝下毒药的生命都会在一星期之后死亡。现在你只有10只小白鼠和1个星期的时间，如何检验出哪个瓶子有毒药？

```
解法：

2的10次方是1024，正好大于1000。

所以这1000瓶药都可以用十位数的二进制数来编号

第1瓶：0000000001

...

第1000瓶：1111110011

10只老鼠分别列成ABCDEFGHIJ

对应十位二进制的每一位

每一瓶药喂给该药对应的十位二进制数为1的老鼠

如第1000瓶喂给：A B C D E F I J号鼠

假设一周后A C H老鼠死了

则毒药的编号是：1010000100对应的十进制数

```

### 烧绳子

现有若干不均匀的绳子，烧完这根绳子需要一个小时，问如何准确计时15分钟，30分钟，45分钟，75分钟

```
15分钟：

- 方法1：一根绳子对折两头烧
- 方法2：拿两根绳子，一根一头烧，一根两头烧，两头烧的那根烧完开始计时
  并且马上点燃原本一头烧绳子的另一头，烧完既是15分钟


30分钟：

- 两头烧

45分钟：

- 同15分钟方法2，从头开始计时

75分钟：

- 30 + 45
```

### 犯人猜颜色

一百个犯人站成一纵列，每人头上随机带上黑色或白色的帽子，各人不知道自己帽子的颜色，但是能看见自己前面所有人帽子的颜色。\
然后从最后一个犯人开始，每人只能用同一种声调和音量说一个字：”黑”或”白”。\
如果说中了自己帽子的颜色，就存活，说错了就拉出去斩了。\
说的答案所有犯人都能听见。\
是否说对，其他犯人不知道。\
在这之前，所有犯人可以聚在一起商量策略。\
问如果犯人都足够聪明而且反应足够快，100个人最大存活率是多少？

```
答案：第一个人只报自己看到的奇偶，50%概率活，剩下的所有人都能活

解法：

1. 最后一个人数前面黑帽子的个数，如果是奇数喊【黑】，偶数喊【白】

2. 其他99个人记住这个值，即黑帽子个数的奇偶，之后每听到一次【黑】就做一次奇偶转换
   之后的人数在自己之前喊了多少个【黑】，再根据第一个人喊的奇偶，记成黑帽子奇偶的现在状态。

3. 如果现在状态的奇偶和自己前排的奇偶不同，则说明自己是黑帽子，喊【黑】，相同喊【白】。

举例：

100号数了前面有45个黑帽子，是奇数，喊【黑】
[喊前全局奇偶状态：无，自己面前：奇，喊后全局奇偶状态：奇]

99号数了前面也是45个黑帽子，是奇数喊【白】，【白】不改变全局奇偶状态
[喊前全局奇偶状态：奇，自己面前：奇，喊后全局奇偶状态：奇]

98号数了前面44个黑帽子，是偶数，喊【黑】，【黑】改变全局奇偶状态
[喊前全局奇偶状态：奇，自己面前：偶，喊后全局奇偶状态：偶]

97号数了前面43个黑帽子，是奇数，喊【黑】，【黑】改变全局奇偶状态
[喊前全局奇偶状态：偶，自己面前：奇，喊后全局奇偶状态：奇]

...
```

假设每个犯人只能看到自己前面人的帽子

```
答案：

1. 所有偶数编号的犯人报自己前面人帽子的颜色，他们50%存活
2. 所有奇数编号100%存活

### 100楼扔鸡蛋问题

有2个鸡蛋，从100层楼上往下扔，以此来测试鸡蛋的硬度。比如鸡蛋在第9层没有摔碎，在第10层摔碎了，那么鸡蛋不会摔碎的临界点就是9层。

问：如何用最少的尝试次数，测试出鸡蛋不会摔碎的临界点？

```
最优解：

逆推法：假设最少尝试X次

1. 首先第一个鸡蛋从X楼开始扔
（只能从X开始，从X+1开始如果碎了总常识次数可能到X+1，从X-1开始不符合X为最优解的假设）

2. 如果碎了用第二个鸡蛋在剩下的X-1楼里面找

3. 如果没碎，为了保证尝试不超过X次，已经用了1次，剩下第一个鸡蛋从X + X-1楼扔

  （1） 如果碎了，第二个鸡蛋找[X, X + X-2]楼
  （2） 如果没碎，往上 X - 2楼再扔第一个蛋

可以推出最优解X满足 x + (x-1) + ... + 2 + 1 = 100

向上取整 X = 14



较优解：

假设每X楼扔一次第一个蛋，那么扔完之后第二个蛋最坏情况要扔 X-1 次

假设运气特别不好，那么需要扔 100 / x + (x - 1)次

如何让这个数最小？ - 让100 / x取 x = 10

没10楼扔一次

```

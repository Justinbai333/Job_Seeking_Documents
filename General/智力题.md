### 1. 抛硬币问题

两人抛硬币，谁先扔出来正面谁赢，假如是你先扔硬币，那么赢的概率是多少？

```
答案：2/3

解答：

假设先扔的一方赢得概率为x，两方总有一个人赢，是必然事件，因此有

x+0.5*x=1，x=2/3

等式的物理意义是我赢的概率(x) 加上我第一次没正面的概率(0.5)，在我失败的情况下对方态势转化为我。
```

### 2. 赛马比赛

25匹马赛跑，5条跑道，跑几次可以知道前3？

```
答案：7次

解答：

1-5次：25匹马分成5组ABCDE，分别跑出最快的A1, B1, C1, D1, E1

6次：A1, B1, C1, D1, E1一起跑，跑出第一名，假设A1

7次：此时第二名很明显只可能是B1或者A2，因此只需要这两个赛跑就可以得到第二名

接着考虑如何用剩下的赛道选出第3：

如果第二名为A2，第三有可能是A3或B1

如果第二名为B1，第三名有可能是A2或者B2或者C1

以下是可能成为2、3名的马匹，正好5匹马，可以分成一组赛跑

[01, A2, A3, --, --]
[B1, B2, --, --, --]
[C1, --, --, --, --]
[--, --, --, --, --]
[--, --, --, --, --]

因此将[A2, A3, B1, B2, C1]一起赛跑可以得到二三名

```

64匹马赛跑，8条跑道，跑几次可以知道前4？

```
答案：11次

解法：

1-8次：64匹马分成8组，分别跑出：A1 ~ H1

9次：A1 ~ H1赛跑，淘汰最后四位：假设最快的是A1 B1 C1 D1

10次：此时一直A1是最快的，可以不用再跑A1，剩下A2 A3 A4 B1 B2 B3 C1 C2 D1
有可能进前4所以第10次从这9个里面抽8个跑，选出前三

11次：第10次跑出来的前三再和剩下的那个跑一次，再选出前三，加上A1就是前四
```

25匹马5条跑道找最快的5匹马，需要跑几次？

```
答案：8-9次

解法：

1-5次：同上，分成5组跑

假设结果：

[A1, A2, A3, A4, A5]
[B1, B2, B3, B4, B5]
[C1, C2, C3, C4, C5]
[D1, D2, D3, D4, D5]
[E1, E2, E3, E4, E5]

6次：同上，每组的第一名比赛

假设结果为A1最快，E1最慢

7次：

根据25匹马 5跑道 选前3，可以知道7轮可以选出前3

8次：

第7次的赛跑情况可能有以下几种结果，此处需要枚举分析所有可能性：

【可能性1：A2第二 A3第三】：

第四可能是 A4 或 B1

第四是A4: 第五可能是A5或B1
第四是B1：第五可能是A4或B2或C1

以下是可能成为4、5名的马，正好5匹，可以一次性决出四五名

[01, 02, 03, A4, A5]
[B1, B2, --, --, --]
[C1, --, --, --, --]
[--, --, --, --, --]
[--, --, --, --, --]

【可能性2：A2第二 B1第三】：

第四可能是A3, B2, C1

假设第四A3，第五可能是A4或B2或C1
假设第四B2, 第五可能是A3或B3或C1
假设第四C1，第五可能是A3或B2或C2或D1

综上：第4、5名需要在马匹[A3、B2、B3、C1、A4、C2、D1]七匹马中产生，则必须比赛两场才行，也就是到第9场角逐出全部的前5名。

【可能性3 - 5以此类推】

结论：需要 8 - 9场可以选出前5

```

### 3. 砝码称轻重

一共有9个砝码，一个天平，已知其中有1个砝码比其他的轻，称几次可以找到这个砝码？

```
答案：2次

解法：

第一次：

先分成3个为一组的3组，任意选两组放在天平上。

如果不一样重，轻的那组里面有要找的砝码。如果一样重，较轻的砝码在剩下的那组里。

第二次：

从找到的组里选两个放在天平上。

如果不一样重，轻的那个是要找的砝码。如果一样重，较轻的砝码是剩下那个。

```

有十组砝码每组十个，每个砝码重10g，其中一组每个只有9g，有能显示克数的秤最少几次能找到轻的那一组砝码？

```
答案：1次

解法：

10组砝码，第一组拿1个，第二组拿2个，以此类推第10组拿10个放在称上。

假设称的读数为 X，550-X就是每个只有9g的那一组
```

### 5. 白鼠与毒药

有1000个一模一样的瓶子，其中有999瓶是普通的水，有1瓶是毒药。任何喝下毒药的生命都会在一星期之后死亡。现在你只有10只小白鼠和1个星期的时间，如何检验出哪个瓶子有毒药？

```
解法：

2的10次方是1024，正好大于1000。

所以这1000瓶药都可以用十位数的二进制数来编号

第1瓶：0000000001

...

第1000瓶：1111110011

10只老鼠分别列成ABCDEFGHIJ

对应十位二进制的每一位

每一瓶药喂给该药对应的十位二进制数为1的老鼠

如第1000瓶喂给：A B C D E F I J号鼠

假设一周后A C H老鼠死了

则毒药的编号是：1010000100对应的十进制数

```

### 8. 100楼扔鸡蛋问题

有2个鸡蛋，从100层楼上往下扔，以此来测试鸡蛋的硬度。比如鸡蛋在第9层没有摔碎，在第10层摔碎了，那么鸡蛋不会摔碎的临界点就是9层。

问：如何用最少的尝试次数，测试出鸡蛋不会摔碎的临界点？

```
最优解：（逆推不是特别符合正常思路，直接回答最优解可能让面试官觉得你见过这道题...）

逆推法：假设最少尝试X次

1. 首先第一个鸡蛋从X楼开始扔
（只能从X开始，从X+1开始如果碎了总常识次数可能到X+1，从X-1开始不符合X为最优解的假设）

2. 如果碎了用第二个鸡蛋在剩下的X-1楼里面找

3. 如果没碎，为了保证尝试不超过X次，已经用了1次，剩下第一个鸡蛋从X + X-1楼扔

  （1） 如果碎了，第二个鸡蛋找[X, X + X-2]楼
  （2） 如果没碎，往上 X - 2楼再扔第一个蛋

可以推出最优解X满足 x + (x-1) + ... + 2 + 1 = 100

向上取整 X = 14，最少需要14次



较优解：

假设每X楼扔一次第一个蛋，那么扔完之后第二个蛋最坏情况要扔 X-1 次

假设运气特别不好，那么需要扔 100 / x + (x - 1)次

如何让这个数最小？ - 让100 / x和（x-1）都尽量小 - 取 x = 10

每10楼扔一次，最坏需要 10 + 9 = 19次

```

### 9. 猴子搬香蕉

一个小猴子边上有100根香蕉，它要走过50米才能到家，每次它最多搬50根香蕉，（多了就被压死了），它每走1米就要吃掉一根，请问它最多能把多少根香蕉搬到家里。（提示：他可以把香蕉放下往返的走，但是必须保证它每走一米都能有香蕉吃。也可以走到n米时，放下一些香蕉，拿着n根香蕉走回去重新搬50根。）

```

答案：16根

解法：

猴子搬香蕉分成两个阶段：

【一阶段】

总香蕉数 > 50 这时候猴子需要往返跑来搬两份香蕉，每份50个

在一阶段的往返中，假设把100个香蕉分装在AB两个盒子里，各50个

每搬一米，猴子需要走三米。

举例：50 - 49米

先搬A一米，在返回一米，然后搬B一米

共走三米，要吃三个香蕉。

【二阶段】

再走了3 * 17 = 51米，吃了51个香蕉后，小猴子离家50 - 17 = 33米

此时AB两个箱子可以合并成一个箱子，还剩下100 - 51 = 49个香蕉。

往后33米每走一米吃一个香蕉，回到家还剩下 49 - 33 = 16个香蕉

```

### 10. 取石子（博弈游戏）

**巴什博弈**

问题：一共有N颗石子（或者其他乱七八糟的东西），每次最多取M颗最少取1颗，A，B轮流取，最后一个拿走石头使剩余石头数量为0的人获胜，问谁最后会获胜？

```
答案：

假设A先拿

情况1. 如果 M > N：
A直接拿N个，A获胜

情况2. 如果 M < N：
如果N可以被M + 1整除，B获胜
如果N不可以被M + 1整除，A获胜


解法：

情况1. Trivial

情况2. 如果N可以被M + 1整除
则不管A一次拿多少个（假设拿X， X <= M），B总可以拿 Y = M + 1 - X个
使剩余石头个数又变成M + 1的倍数。

直到最后两轮，B拿完让石头数量剩下M + 1，A不管拿几个都输。

如果N不可以被M + 1整除，则A在第一轮拿Z个，Z为 N / M+1 的余数
之后流程同上，只是AB位置互换，B输。

```

### 12. 海盗分金币

5个海盗抢到了100枚金币，每一颗都一样的大小和价值。

他们决定这么分：

抽签决定自己的号码（1，2，3，4，5）\
首先，由1号提出分配方案，然后大家5人进行表决，\
当半数以上的人同意时（ 不包括半数，这是重点），按照他的提案进行分配，否则将被扔入大海喂鲨鱼。 \
如果1号死后，再由2号提出分配方案，然后大家4人进行表决，当且仅当半超过半数的人同意时，按照他的提案进行分配，否则将被扔入大海喂鲨鱼。\
依次类推......


假设每一位海盗都足够聪明，并且利益至上，能多分一枚金币绝不少分，那么1号海盗该怎么分金币才能使自己分到最多的金币呢？

```
答案：(97, 0, 1, 2, 0)或者(97, 0, 1, 0, 2)

逆推法

- 假设1 2 3号都喂鲨鱼了，只剩4 5号，4号提出任何方案5号都会否决

- 所以4号为了保命，会支持3号的任何提议。所以3号的提议可以是(0, 0, 100, 0, 0)

- 2号知道了3号的决策后，为了让自己不被喂鲨鱼，会放弃三号的投票，提出给4号和5号1枚金币。
即他的提议为(0, 98, 0, 1, 1), 四号和五号因为可以多拿金币会支持2号而不让他喂鲨鱼

- 同理1号知道了2号的决策后，会放弃2号的投票，给3号1枚金币，给4号或者5号2枚金币。
此时3号会因为自己多拿了一枚金币支持1号，拿了两枚金币的4号或者5号也会支持1号。
所以1号的提议会是(97, 0, 1, 2, 0)或者(97, 0, 1, 0, 2)
```
